# Text RPG Client

文字 RPG 放置游戏 - 前端

## 🎮 在线游玩

GitHub Pages: https://hupose.github.io/text-rpg-client

## 📦 本地开发

```bash
# 克隆仓库
git clone https://github.com/hupose/text-rpg-client.git
cd text-rpg-client

# 启动本地服务器
python3 -m http.server 8080

# 浏览器访问
# http://localhost:8080
```

## 🔗 连接后端

1. 打开页面
2. 输入后端服务器地址（如 `https://your-server.com`）
3. 点击「连接服务器」
4. 创建角色开始游戏

## 🛠️ 技术栈

- 纯 HTML/CSS/JavaScript
- WebSocket (Socket.IO Client)
- 无需构建，直接运行
